🚗 Car Dealer Sales Dashboard (Excel)
📌 Overview

This project focuses on analyzing car sales data from January to December 2024 using Microsoft Excel. The goal is to extract meaningful insights from raw data and present them through an interactive dashboard for better business decision-making.

🎯 Objective
Analyze sales performance across different regions, brands, and car types
Identify revenue trends and high-performing segments
Build a dashboard to enable quick and data-driven insights

🛠️ Tools & Technologies
Microsoft Excel
Pivot Tables
Pivot Charts
Slicers & Filters
Data Cleaning Techniques

📊 Key Actions
Cleaned and structured 1000+ rows of sales data
Created pivot tables to summarize Units Sold and Revenue
Built an interactive dashboard with slicers for Region, Brand, and Month
Designed visualizations to track monthly trends and performance comparison

📈 Key Insights
Identified the top-performing region contributing ~30%+ of total revenue
Found the highest-selling car segment based on units sold
Detected peak sales months, indicating seasonal trends
Compared performance across brands to highlight market leaders

🚀 Outcome
The dashboard simplifies complex data into clear visuals, helping stakeholders make data-driven decisions, improve sales strategies, and identify growth opportunities.
